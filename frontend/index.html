Segurança dos dados

Backups e exportação

☁️

Um backup automático é criado a cada 24h enquanto o servidor está rodando, e sempre antes de qualquer importação ou restauração — então dá pra desfazer se algo sair errado. Os arquivos ficam em backend/data/backups/. Isso protege contra erros, mas continua tudo no mesmo disco: baixe uma cópia de vez em quando e guarde em outro lugar (nuvem, pendrive) pra se proteger de perda do disco/computador.

Nenhum backup ainda. Clique em Fazer backup agora para criar o primeiro, ou deixe o servidor rodando por 24h para o backup automático entrar em ação.
